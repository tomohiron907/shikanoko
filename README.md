# Shikanoko

しかのこのこのここしたんたんのプログラミング言語

ほとんどbrainfuck言語

```
    '>': 'こし'
    '<': 'たん'
    '+': 'しか'
    '-': 'ぬん'
    '.': 'のこ'
    ',': 'なら'
    '[': 'つの'
    ']': 'のつ'
```


# install
```
g++ -o shikanoko main.cpp
./shikanoko hello_world.shknk
```

拡張子は.shknkです。

command_converter.pyは、brainfuckの文字列を.shknkに変換するpython コードです。(.shknkを書くのが面倒なので)
